# Quick Deployment Guide

## Option 1: Surge.sh (Fastest - 2 minutes)

1. Open terminal in the `Fest-Website` folder
2. Run: `cd build`
3. Run: `surge`
4. Enter email and password (creates free account)
5. Press Enter to use suggested domain or type your own
6. Your site will be live instantly!

**Your site will be at:** `https://your-domain.surge.sh`

## Option 2: Netlify Drop (No CLI needed - 1 minute)

1. Go to: https://app.netlify.com/drop
2. Drag and drop the `build` folder
3. Your site is live instantly!
4. You'll get a URL like: `https://random-name.netlify.app`

## Option 3: Vercel (Fast - 2 minutes)

1. Go to: https://vercel.com
2. Sign up/login (free)
3. Click "New Project"
4. Drag and drop the `build` folder
5. Deploy!

## Option 4: GitHub Pages (If you have GitHub)

1. Create a GitHub repository
2. Upload the `build` folder contents
3. Go to Settings > Pages
4. Select main branch
5. Your site will be at: `https://yourusername.github.io/repo-name`

---

**Recommended: Use Netlify Drop (Option 2) - It's the fastest and easiest!**
